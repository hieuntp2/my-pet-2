# Animation Timing Rules

## Blink

Recommended timing:

- close duration: 70–120 ms
- closed hold: 20–60 ms
- open duration: 100–180 ms
- interval: random 1800–5200 ms

Never use a perfectly fixed interval.

## Idle Breathing / Floating

Recommended timing:

- cycle: 2500–5000 ms
- vertical movement: very small
- scale/squash: very small
- should be visible only when observed

## Look Around

Recommended timing:

- every 4–12 seconds during idle
- movement duration: 400–900 ms
- hold duration: 300–1200 ms
- return duration: 400–900 ms

## Tap Reaction

Recommended timing:

- total duration: 350–800 ms
- quick anticipation or surprise
- return to base

## Long Press Reaction

Recommended timing:

- total duration: 600–1200 ms
- can be affectionate or annoyed later

## Feed/Play/Rest Reactions

Recommended timing:

- feed: 700–1200 ms
- play: 600–1100 ms
- rest: 800–1500 ms

## Cooldown

- tap cooldown: 250–500 ms
- repeated reaction softening after 3 rapid taps
- do not ignore all input silently; show softer response later

## Anti-Repeat

Avoid playing the same exact reaction more than twice in a row.

If repeated:

- choose alternate reaction
- reduce intensity
- switch to annoyed/curious based on state later
